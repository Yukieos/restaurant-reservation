Restaurant Reservation System

A comprehensive web application for restaurant reservations in New York

About the Project

This project is a full-featured restaurant reservation system that allows users to browse restaurants in New York, make reservations, and manage their bookings.

User Accounts

Create an account, log in, and manage your profile and payment details.

Restaurant Discovery

Browse and search for restaurants in New York with details on menus and reviews.

Reservation Management

Make, view, and manage your restaurant reservations with ease.

Technical Details

This application is built with:

  • Python Flask for the backend
  • SQLAlchemy for database interactions
  • PostgreSQL database for data storage
  • HTML/CSS for the frontend
